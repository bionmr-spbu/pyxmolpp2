pyxmolpp2.geometry.calc_autocorr_order_2(vectors, limit= -1) → List[float]¶

Calculates vector auto correlation function \(C(t)\) of second order

\[\begin{split}C(t) = \frac{4 \pi}{5} \sum_{m=-2}^{m=2} \left\langle Y^{2,m}(\theta(\tau),\phi(\tau)) \overline{Y^{2,m}(\theta(\tau+t),\phi(\tau+t))} \right\rangle_\tau \\ = \left\langle P_2\left(\cos(\gamma_{\tau,\tau+t})\right) \right\rangle_\tau\\ = \left\langle \frac{1}{2}\left(3 \cos^2(\gamma_{\tau,\tau+t}) - 1\right) \right\rangle_\tau\end{split}\]
Parameters

  • vectors (VectorXYZ) – list of not normalized vectors

  • limit (int) – strip output to first limit points. If negative return list size match size of vectors

Return type

List[float]

pyxmolpp2.geometry.calc_autocorr_order_2_PRE(vectors, limit= -1) → List[float]¶

Calculates vector auto correlation function \(C(t)\) of second order. Spherical harmonics are multiplied by \(r^{-3}\)

\[\begin{split}C(t) = \frac{4 \pi}{5} \sum_{m=-2}^{m=2} \left\langle Y^{2,m}(\theta(\tau),\phi(\tau)) r(\tau)^{-3} \overline{Y^{2,m}(\theta(\tau+t),\phi(\tau+t)) r(\tau+t)^{-3}} \right\rangle_\tau \\ = \left\langle P_2\left(\cos(\gamma_{\tau,\tau+t})\right) r(\tau)^{-3} r(\tau+t)^{-3} \right\rangle_\tau\\ = \left\langle \frac{1}{2}\left(3 \cos^2(\gamma_{\tau,\tau+t}) - 1\right) r(\tau)^{-3} r(\tau+t)^{-3} \right\rangle_\tau\end{split}\]
Parameters

  • vectors (VectorXYZ) – list of not normalized vectors

  • limit (int) – strip output to first limit points. If negative return list size match size of vectors

Return type

List[float]

pyxmolpp2.geometry.calc_sasa(coordinates, vdw_radii, solvent_radius, indices_of_interest, n_samples= 20)

Calculates per atom Solvent Accessible Solvent Area (SASA) using Lee & Richards’ algorithm

Parameters

  • coordinates (VectorXYZ) – list of atom coordinates

  • vdw_radii (buffer) – list of atom VDW radii

  • solvent_radii – radius of solvent (1.4 is common value for water)

  • n_samples (int) – number of atom slices (default=20). linearly increases complexity

  • indices_of_interest (array) – limit calculation of SASA to points with given indices. If None calculate for all points

  • solvent_radius (float) – not documented yet

Returns

List of per-atom exposed area (0 for fully buried atoms, \(4\pi R_{vdw}^2\) for isolated atoms)

Return type

array
